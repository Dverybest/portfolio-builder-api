# Portfolio Builder API
## Description

Api for CRV portfolio builder application.

## Tech Stack

- NestJs (NodeJS)
- MongoDB

## Installation

```bash
$ yarn install
```

## Running the app

create and set up the .env file as specified in the env-example file located in the root directory.

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
